# codex-da-vida
Codex da Vida — Jogo educativo de Biologia (React/JSX) com expedição pelos 6 biomas brasileiros. Taxonomia, nomenclatura binomial, critérios de classificação e ecologia. 33 questões, atividades interativas, sistema de pontuação e certificado. Inclui versão AEE acessível.
